dash-upset

Interactive UpSet plots for Plotly Dash — in one import.

Set-intersection analysis that stays readable where Venn diagrams collapse. Feed it set memberships and get a matrix, set-size bars, and intersection-size bars: as a plain go.Figure for notebooks and static export, or a self-wiring Dash component. Pure Python, MIT, dataframe-agnostic.

What is an UpSet plot?

An UpSet plot visualizes the intersections of many sets. Venn and Euler diagrams become unreadable past three or four sets; UpSet replaces the overlapping circles with a compact, sortable matrix plus aligned bar charts.

▦

The matrix

Columns are intersections, rows are sets. Filled connected dots show which sets participate in each intersection.

📊

Intersection-size bars

The cardinality of every intersection, on a common scale, so the big overlaps are obvious at a glance.

📏

Set-size bars

The total size of each set, aligned to the matrix rows.

Quick start

Drop the UpSet component into a Dash layout and hand it a dataframe with one boolean column per set. Clicks surface as component properties you read the standard Dash way: Input(id, property).

import pandas as pd
from dash import Dash, Input, Output, callback, html
from dash_upset import UpSet

# One row per misclassified test example; 1 = that model got it wrong.
# The overlaps are the shared hard cases; the singletons are each
# model's private blind spots.
df = pd.DataFrame(
    {
        "ResNet": [1, 1, 0, 1, 0, 1],
        "ViT": [1, 1, 1, 0, 0, 1],
        "XGBoost": [0, 1, 1, 1, 1, 0],
    }
)

app = Dash(__name__)
app.layout = html.Div(
    [
        UpSet(
            id="errors",
            data=df,
            sets=["ResNet", "ViT", "XGBoost"],
            sort_by="cardinality",  # or "degree", "deviation"
            mode="distinct",  # or "intersect", "union"
            theme="auto",  # follows the page (light / dark)
        ),
        html.Pre(id="out"),
    ]
)


@callback(Output("out", "children"), Input("errors", "selected_intersection"))
def show(selection):
    # {"label": "ResNet & ViT", "sets": ["ResNet", "ViT"], "size": 2}
    return str(selection)


if __name__ == "__main__":
    app.run(debug=True)

Just the figure

Need a plain Plotly figure for a notebook, static export, or your own dcc.Graph? create_upset takes the same input and returns a go.Figure.

from dash_upset import create_upset

fig = create_upset(df, sets=["ResNet", "ViT", "XGBoost"], title="Model error sets")
fig.show()  # or dcc.Graph(figure=fig); export to PNG/SVG/PDF via kaleido

Other input shapes

Pre-aggregated counts, per-set contents, or per-element memberships — the familiar upsetplot conventions, reimplemented and dataframe-agnostic via narwhals. Pass any of these as data=.

from dash_upset import from_counts, from_memberships, from_contents

from_counts({"Action": 320, "Action&Comedy": 84})  # pre-aggregated sizes
from_memberships([("A",), ("A", "B"), ()])           # per-element set names
from_contents({"A": ["x", "y"], "B": ["y", "z"]})    # per-set element ids

What's in the box

🔗

Intersection modes

distinct (exclusive), intersect (inclusive), and union counting — the analytic knob most tools omit.

↕️

Sorting

By cardinality, degree, or deviation, with deterministic tie-breaking. Sets by size or name.

🔍

Filtering

Minimum and maximum subset size and degree, top-N intersections, and hide-empty.

📉

Deviation

How surprising each intersection is versus independence — on hover and as a sort key.

🖼️

Publication export

It's a Plotly figure: interactive in the browser, or static PNG/SVG/PDF via kaleido.

🧰

Dataframe-agnostic

Indicator tables from pandas, Polars, PyArrow, and more, through narwhals — no pandas dependency.
